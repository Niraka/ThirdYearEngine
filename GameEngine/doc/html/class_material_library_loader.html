bool MaterialLibraryLoader::loadMaterialLibrary ( std::string  sFile,
std::shared_ptr< MaterialLibrary >  lib 
)

Loads a material library from an .mtl file. If the file could not be found, this function does nothing. Note that the file should be given excluding its fixed parent directory as this is automatically prepended. For example, ".\\data\\materials\\my_mat.mtl" should be given as "my_mat.mtl".
