Detailed Description

A dependency allows a class to attach a list of dependencies to itself. This list is then read each time a new module or service becomes available and is used to notify the class of the new module or service.

Adds a dependency to this module. When the given module becomes available, it will be presented to this module for processing. Note that dependencies cannot be removed as this does not make any sense. Furthermore, a module should not be a dependency of itself.
