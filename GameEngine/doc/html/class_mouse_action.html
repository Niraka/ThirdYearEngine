Detailed Description

A MouseAction is an interface class that provides the necessary functionality for a class to be capable of executing instructions in response to a mouse input event.

Note that while it is perfectly acceptable to use an InputListener in place of a ControlSet and its associated Key/MouseActions's, the latter is preferable in situations where a large number of controls need to be bound/unbound at one time and from one location (typically the Scene).
