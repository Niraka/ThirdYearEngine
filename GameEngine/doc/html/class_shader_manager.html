Detailed Description

The ShaderManager is a component of the WindowModule and is responsible for the production and storage of Shader objects.

Only one shader may be activated at any time. Note that when a shader is deactivated, the uniform values it was last provided remain intact.

bool ShaderManager::createShaders ( std::string  sShaderName)

Creates a Shader with the given name. If any of the shaders fail to load or compile, or the name was already in use, this function will fail. Note that the shader name is also the name of the file that the shaders will be loaded from once the appropriate file extensions have been appended. This has been done to enforce naming consistency. Example usage: Shader name: "skybox" Vertex shader: ".\\data\\shaders\\skybox.vert" Fragment shader: ".\\data\\shaders\\skybox.frag"
